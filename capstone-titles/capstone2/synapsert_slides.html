Slide 2 · The Problem
The Scene & Pain

"Two hours of brainstorming. One messy notebook. Half the ideas — already lost."

Cognitive Overload

Passive transcription tools dump flat text. No visual structure, no conceptual relationships, and no semantic contradictions are preserved.

Processing Latency

Cloud-dependent API configurations impose 3–10s delays. By the time the system parses, the conversation has moved on.

Context Collapse

After 60 minutes of multi-party debate, system context windows overflow, forgetting initial foundational project premises.

Slide · Objectives
Research Objectives

What SynapseRT Sets Out to Achieve

Build a sub-700ms real-time transcription pipeline

Design and implement a local edge ingestion layer using Silero VAD and Whisper.cpp that processes streaming audio with end-to-end latency under 700ms.

Orchestrate 4-agent coherent knowledge construction

Deploy a multi-agent system with MESI-inspired cache coherence across Context-Capture, Arbitrator, Suggestive-Prompt, and Corrective agents running in parallel.

Implement a 3-tier hardware-inspired memory hierarchy

Construct a tiered memory model (L1 context window, L2 Redis semantic cache, LT Neo4j graph) to prevent context collapse during extended multi-party sessions.

Validate with empirical measurement

Evaluate cognitive load reduction using NASA-TLX and semantic precision using R-graph scoring against baseline flat-text tools.

Deliver an interactive D3.js knowledge graph

Produce a SvelteKit-based visualization dashboard rendering the evolving relational knowledge map in real time as discussions progress.

Slide 6 · Execution & Trust
Project Feasibility

Difficulty Acknowledgment & Roadmap

Taglish STT Accuracy
Whisper's phonetic performance under high-speed code-switching (Taglish) is highly variable. We mitigate this using custom prompt-priming arrays and phonetic mapping hooks.
Lock-Free Agent Concurrency
Preventing deadlocks when 4 agents write to the same L2 Redis cache context window demands strict implementation of the MESI-inspired cache coherence rules.
D3 Layout Jitter
Streaming data insertion causes node layout physics loops to jump erratically. We resolve this by tuning custom alpha-decay loops and locking older nodes.
